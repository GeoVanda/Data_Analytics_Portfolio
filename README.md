# Project: Energy Transition: From Global to EU27 (2025) 

# Context 
This project analizes global and regional trends in renewable electricity generation as part of the broader energy transition process with a focus on the EU27 and Spain. It presents a variety of visuals and findings based on an interactive dashboard developed on Power BI.

## Datasets
- IRENA Energy Statistics (2025) - Data Source: https://www.irena.org/Publications/2025/Jul/Renewable-energy-statistics-2025
- World Bank Population (by country) - Data Source: "Indicadores del desarrollo mundial", 2025
- World Bank GDP (by country) - Data Source: "Indicadores del desarrollo mundial",
- ISO country codes

- Data transformations have been implemented directly on PowerBI & are included in `Projects/Energy Transition From Global to EU27 (2025) - Charts selection-2025.pbix`

- Screenshots of the dashboard are available in the `images/` folder.

## How to Use
